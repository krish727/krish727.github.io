How to Get Data from Table by Partition (ORACLE ONLY)

Mon 09 November 2015

How to Specify Partition in the Select statement of the Oracle Tables

select * from TABLE_NAME partition( PARTITION_NAME)

How to get all the paritions in a table or all tables

SELECT * FROM dba_tab_partitions WHERE table_name = 'TABLE_NAME';
