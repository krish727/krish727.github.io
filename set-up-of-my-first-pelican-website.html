Set up of my first Pelican website

Mon 30 November 2015

Python 2.7.x in virtualenv with pyenv

After an initial hiccup in setting up of the virtualenv environment, I read the trend is more towards pyenv and other wrappers around it. I ended up using this blog to setup pyenv with python 2.7.10.

Installation/Setup of Pelican in virtual environment

Once the virtual environment issue was resolved, now it was time to install pelican. Installation itself was a breeze using the standard installation document located here.

After installing pelican, I installed the other supporting packages that are mentioned in the link and went ahead and did the pelican-quickstart command to do a quick start of the pelican setup.

Next step was to follow this link to create some basic pages to have a site up an running at the local webserver.

Now I wanted to customize my website with the themes for pelican using the instructions that were provided here. After some experimentation with various themes, I settled on the octopress theme (which might change in future) which is what this page is made with.

Finally I wanted to host this page at GitHub static page hosting site. Here I encountered the famous 403 problem. I used this blog to get over the 403 problem.

It took me sometime to research all the things that I noted here but was impressed with all the things that are there to help out in creating a good looking website without much of design experience. Hopefully I will do some more experimentations with pelican and post my findings here.

Till then..
